ℹ️ How to Use This Visualization

  • Attribute Selection: Use the tab buttons to switch between attributes.
  • Histogram: Hover over bars to see value ranges and count of counties.
  • Choropleth Map: Displays counties colored based on the selected attribute (e.g., Population with no health insurance%). Hover to view detailed info.
  • Legend: Located below the choropleth map. It shows the color gradient corresponding to attribute values.
  • Scatterplot: Compares two selected attributes.
  • Tooltip: Appears on hover for detailed data, shows data based on selected attribute (e.g., Population with no health insurance, Population with no education).
  • Interactive Brushing: Use the brush tool on the Scatterplot to filter counties on the Choropleth Map. Dynamically updates the Choropleth Map and Histogram to focus only on selected data ranges.
